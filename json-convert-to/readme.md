Los json de la carpeta input deben de tener la forma

```
{
    key : value,
    key1 : value1,
    key2 : value2,
    key3 : value3,
    key4 : value4,
}
```

Al ejecutar este programa, se devolvera en la carpeta output, jsons de la forma

```
[
    value,
    value1,
    value2,
    value3,
    value4,
]
```
